# 免费订阅

《前端小课》虽然已经写了 1 年，我毅然决定要重写，主要原因是以下几点：

- 写的内容不连贯，不够系统全面；
- 有些内容写的不是很好，不够通俗易懂；
- 受公众号限制，代码不能够直接运行，纠错很难，各个章节跳转受限；
- 网站更能够表现出「多媒体」电子书这一特点；

这次正在重写《前端小课》，由于业余时间有限，大约需要花费 2-3 年的时间。以下内容来源于我的公众号：
